hapmap_am368 {IntAssoPlot}R Documentation

Genotype file.

Description

A dataset containing the genotype file. The variables are as follows:

Usage

hapmap_am368

Format

:

rs

contains the SNP identifier.

alleles

contains SNP alleles according to NCBI database dbSNP.

chrom

contains the chromosome that the SNP was mapped.

pos

contains the respective position of this SNP on chromosome.

strand

contains the orientation of the SNP in the DNA strand. Thus, SNPs could be in the forward (+) or in the reverse (-) orientation relative to the reference genome.

assembly

contains the version of reference sequence assembly (from NCBI).

center

contains the name of genotyping center that produced the genotypes.

protLSID

contains the identifier for HapMap protocol.

assayLSID

contain the identifier HapMap assay used for genotyping.

panelLSID

contains the identifier for panel of individuals genotyped.

QCcode

contains the quality control for all entries.

subsequently, the list of sample names.
